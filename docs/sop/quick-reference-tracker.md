# Quick reference — calibration tracker (operators)

**No admin tasks** — day-to-day use only.

## Start a job

1. **Performing date & time** → set  
2. **Vehicle** → pick from list, or **Others** + type name  
3. **Owner** → pick your name  
4. **Start calibration**

## During the job

| Goal | Action |
|------|--------|
| Move forward | **Next** (on last step: **Finish**) |
| Undo one step | **Back** (not on first step) |

## After Finish

Row moves to **Completed calibrations** (read-only).

## Remember

- Page auto-updates ~every **4 seconds** and when you **return to the tab**.  
- Same list for **everyone** — coordinate if two people edit one vehicle.
